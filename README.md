# Intelligent Assistant and Advisory System for Online Stores
# ผู้ช่วยและให้คำปรึกษาอัจฉริยะสำหรับร้านค้าออนไลน์


   แอปพลิเคชัน Streamlit นี้เป็นระบบวิเคราะห์และแนะนำการจัดสเปคคอมพิวเตอร์ตามลักษณะการใช้งานของผู้ใช้ โดยพัฒนาขึ้นเพื่อช่วยให้ผู้ใช้งานสามารถเลือกอุปกรณ์คอมพิวเตอร์ได้เหมาะสมกับความต้องการและงบประมาณของตนเอง
ระบบจะดึงข้อมูลสเปคและราคาของอุปกรณ์จากอินเทอร์เน็ต (Web Scraping) แล้วนำมาวิเคราะห์ร่วมกับข้อมูลการใช้งานที่ผู้ใช้ระบุ เช่น ใช้ทำงานเอกสาร เล่นเกม ตัดต่อวิดีโอ หรือทำงานกราฟิก จากนั้นจะแนะนำสเปคที่เหมาะสมและคุ้มค่าที่สุด
กลุ่มเป้าหมายคือ นักเรียน นิสิต นักศึกษา และบุคคลทั่วไปที่ต้องการจัดสเปคคอมพิวเตอร์ แต่ยังขาดความรู้ด้านเทคนิคหรือไม่แน่ใจว่าจะเลือกอุปกรณ์อย่างไร
แอปนี้ช่วยแก้ปัญหาการเลือกสเปคที่ไม่เหมาะสม เช่น เลือกสเปคต่ำเกินไปจนใช้งานไม่ลื่น หรือสูงเกินความจำเป็นจนสิ้นเปลืองงบประมาณ ทำให้การตัดสินใจซื้อคอมพิวเตอร์มีความแม่นยำ ประหยัด และมีประสิทธิภาพมากขึ้น

## 🚀 ฟีเจอร์หลัก (Features)
1. การวิเคราะห์ข้อมูลร้านค้า (Store Data Analysis)
ระบบสามารถรับข้อมูลร้านค้า เช่น
- ยอดขายย้อนหลัง
- ข้อมูลสินค้า
- ข้อมูลสต็อก
แล้วนำมาวิเคราะห์ด้วย AI เพื่อดูแนวโน้มยอดขาย หรือปัญหาที่เกิดขึ้นในร้านค้า

2. ระบบให้คำปรึกษาและคำแนะนำทางธุรกิจ (Business Advisory System)
ผู้ใช้สามารถถามคำถามหรือขอคำแนะนำเกี่ยวกับธุรกิจ เช่น
- วิธีเพิ่มยอดขาย
- วิธีจัดการสต็อกสินค้า
- วิธีสื่อสารกับลูกค้า
ระบบจะประมวลผลและให้คำแนะนำที่เข้าใจง่าย เหมือนมี ผู้ช่วยธุรกิจส่วนตัว

3. การแสดงผลและสรุปข้อมูลผ่าน Web Application (Web Dashboard)
ระบบจะแสดงผลลัพธ์การวิเคราะห์ในรูปแบบที่เข้าใจง่าย เช่น
- ข้อความสรุป
- กราฟข้อมูล
- รายการคำแนะนำ
เพื่อช่วยให้ผู้ใช้สามารถนำข้อมูลไปใช้ตัดสินใจทางธุรกิจได้ทันที
